# excel-data-cleaner

Excel Data Cleaner

A simple Python project that cleans raw Excel/CSV data using Pandas.

Features

- Removes duplicate rows
- Handles missing values
- Converts date columns into proper format
- Exports a cleaned CSV file

Tech Used

- Python
- Pandas

How to Run

1. Install dependencies:

pip install pandas

2. Run the script:

python clean_data.py

Output

Generates a new file:

cleaned_sales.csv
